## Problem

Manual monitoring of indicators and trade execution
was time-consuming and error-prone.

## Solution

Built an automated pipeline that continuously processes
market data, calculates indicators and monitors trade conditions.

## Features

- Real-time OHLC generation
- EMA, RSI, MACD, VWAP indicators
- PostgreSQL / Supabase integration
- Trade logging
- Active trade management
- Telegram notifications
- Stop-loss & take-profit handling

- ## Tech Stack

Python, PostgreSQL, Supabase,
Flask, REST APIs, AWS, GitHub

Core trading strategy logic intentionally omitted.
Repository focuses on architecture, workflows,
system design and implementation overview.
